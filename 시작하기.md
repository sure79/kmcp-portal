# KMCP 업무포털 실행 가이드

## 사전 준비 (최초 1회)

1. **Node.js 설치** (없는 경우)
   - https://nodejs.org 에서 LTS 버전 다운로드 후 설치

2. **패키지 설치**
   - 이 폴더에서 터미널(cmd) 열기
   - 아래 명령어 실행:
   ```
   npm install
   ```

## 서버 시작

```
npm start
```

또는 개발 모드 (파일 변경 자동 반영):
```
npm run dev
```

## 접속 방법

- **서버 컴퓨터에서**: http://localhost:3000
- **다른 PC에서**: http://[서버IP]:3000
  - 서버 IP 확인: cmd에서 `ipconfig` 입력 후 IPv4 주소 확인

## 기본 계정

| 항목 | 값 |
|------|-----|
| 아이디 | admin |
| 비밀번호 | admin1234 |

> ⚠️ 처음 로그인 후 비밀번호를 변경하세요!

## 주요 기능

| 메뉴 | 기능 |
|------|------|
| 대시보드 | 오늘 일정, 보고 현황, 회의 알림 |
| 팀 현황판 | 전체 직원 업무 한눈에 보기 |
| 업무보고 | 일일 보고서 작성/조회 (월~금) |
| 칸반 보드 | 드래그로 작업 진행상태 변경 |
| 공통 프로젝트 | 공동 프로젝트 진행상황 관리 |
| 회의 관리 | 주간회의(월)/기술회의(목) 관리 |
| 공지사항 | 사내 공지 게시 |
| 사용자 관리 | 직원 추가/수정/삭제 (관리자만) |

## 정기 회의 알림

- **주간회의**: 매주 월요일 08:30 → 대시보드에 자동 표시
- **기술회의**: 매주 목요일 10:00~12:00 → 대시보드에 자동 표시
